# 📄 Smart Resume Builder

A modern, responsive resume builder that runs entirely in your browser — no backend, no sign-up, no paid APIs.

## ✨ Features

- **Personal Info** — Full name, email, phone, job title, professional summary
- **Education** — Add multiple education entries dynamically
- **Skills** — Comma-separated skill tags
- **Work Experience** — Add multiple entries with descriptions
- **Live Preview** — Generates a clean, print-ready resume on the page
- **Download** — Downloads your resume as a standalone HTML file
- **Responsive** — Looks great on mobile, tablet, and desktop
- **Dark Mode UI** — Premium dark interface with vibrant accents

## 🚀 Getting Started

### Prerequisites

No build tools are required. This is a static site.

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/smart-resume-builder.git
   cd smart-resume-builder
   ```

2. **Open in browser**
   Simply open `index.html` in your browser:
   ```bash
   # macOS
   open index.html

   # Windows
   start index.html

   # Linux
   xdg-open index.html
   ```

   Or use a local dev server:
   ```bash
   npx serve .
   ```

## 🌐 Deployment

### Vercel (Recommended)

1. Push this repo to GitHub.
2. Go to [vercel.com](https://vercel.com) and import the repository.
3. Vercel will auto-detect it as a static site — no configuration needed.
4. Click **Deploy**. Done!

### GitHub Pages

1. Push to GitHub.
2. Go to **Settings → Pages**.
3. Select the `main` branch and `/root` folder.
4. Your site will be live at `https://your-username.github.io/smart-resume-builder/`.

### Netlify

Drag and drop the project folder onto [app.netlify.com/drop](https://app.netlify.com/drop).
